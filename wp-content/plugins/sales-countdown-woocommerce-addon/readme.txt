== Description ==
We have developed add-on for WooCommerce plugin that enhances the current feature to next level
This plugin is used to start the countdown for specified product. Countdown products will be displayed as a carousel.

* Use below shortcode in post/page editor to show sales countdown product carousel

`
 [sales-countdown-product]
`

You can also put shortcode in template file as follow:
  
`
<?php echo do_shortcode('[sales-countdown-product]');?>
`

== Installation ==
1. Upload the folder "sales-countdown-for-woocommerce" to "/wp-content/plugins/".

2. Activate the plugin through the "Plugins" menu in WordPress.

3. To add Sales countdown product, go to admin panel and open any woocommerce product, You will find a meta box "Sales Countdown Box" in the right side. There are 5 sections in "Sales Countdown Box".	
	* Add Product in Sales Countdown: You can enable or disable current product as a countdown product.
	* Date: Choose a date from the date of the current date.
	* Hour: Select hour.
	* Minute: Select minute.
	* Second: Select second.
	
4. Manage carousel settings by going through "Dashboard -> Sales Countdown".
5. Use below shortcode in post/page editor to show sales countdown product carousel.

`
 [sales-countdown-product]
`

6) You can also put shortcode in template file as follow:

`
<?php echo do_shortcode('[sales-countdown-product]');?>
`
